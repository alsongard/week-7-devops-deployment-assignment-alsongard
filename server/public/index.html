BugTracker API

Version 1.0 — RESTful API Endpoints Overview

Bug Endpoints

  • GET /bugs — Fetch all bugs
  • GET /bug/:id — Fetch a bug by ID
  • GET /getUserId/:id — Get bugs reported by a specific user
  • PUT /bug/:id — Update a bug by ID
  • DELETE /bug/:id — Delete a bug by ID

User Endpoints

  • POST /register — Register a new user
  • POST /login — Login as a user

Admin Endpoints

  • POST /admin — Create an admin account
  • POST /logadmin — Login as admin
  • GET /users — Fetch all registered users
